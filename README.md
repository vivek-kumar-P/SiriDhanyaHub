# Siri-Dhanya Hub

Android internship project scaffold for the Siri-Dhanya Hub app.

## What is included
- Android project structure
- Gradle Kotlin DSL setup
- Jetpack Compose configuration
- Hilt, Room, Navigation, Firebase Auth dependency declarations
- Basic app theme and starter screen
- Execution plan and setup notes

## Current status
- Project files are scaffolded locally
- Machine-level Android dependencies may still need to be installed
- Gradle is pinned to `8.7` for Android Gradle Plugin `8.5.2`

## Next implementation order
1. Install JDK and Android SDK tools
2. Open project in Android Studio
